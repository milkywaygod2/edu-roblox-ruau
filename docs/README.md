# docs 문서 네이밍 컨벤션

이 폴더의 문서는 아래 네이밍 규칙을 따릅니다.

## 형식

```
{CATEGORY}_{영역}_{세부}.md
```

- **CATEGORY**: 대문자 분류 (rule-git 커밋 TYPE 계열 차용)
- **영역**: 한글 도메인/대상 (예: `루아우`, `로블록스`, `공통모듈`, `커리큘럼`)
- **세부**: 한글 제목, 공백 없이 붙여쓰기

구분자는 언더스코어(`_`), 확장자는 `.md`.

## 카테고리

| CATEGORY | 용도 | 예시 |
|----------|------|------|
| `ARCH`   | 아키텍처·구조 설계 | `ARCH_공통모듈_개념별분리.md` |
| `FEAT`   | 기능 기획·구상 | `FEAT_루아우_도입계획.md`, `FEAT_로블록스_스크립트동기화.md` |
| `GUIDE`  | 강의·교육 자료 | `GUIDE_커리큘럼_12주차.md`, `GUIDE_루아우_강의가이드.md` |

> 새 카테고리가 필요하면 대문자 한 단어로 추가하고 이 표에 등록합니다.

## 규칙

1. `CATEGORY`는 **대문자 한 단어**로 시작한다.
2. 한글 제목은 **공백 대신 붙여쓰기**한다. (영문 고유명사 혼용 허용: `Luau`, `Git`, `n8n`)
3. 각 파트는 언더스코어(`_`)로만 구분한다.
4. **예외**: `README.md`와 외부 산출물 디렉터리(`roblox_api_dump/` 등)는 원래 이름을 유지한다.

## 예시 (현재 docs)

```
ARCH_공통모듈_개념별분리.md
FEAT_로블록스_스크립트동기화.md
FEAT_루아우_검증도구설치.md
FEAT_루아우_도입계획.md
GUIDE_커리큘럼_12주차.md
GUIDE_루아우_강의가이드.md
```
